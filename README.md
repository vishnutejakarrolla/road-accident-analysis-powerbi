# 🚀 Road Accident Analysis (Power BI)

## 📊 Project Overview
This project analyzes road accident data using Power BI to identify trends and patterns.

## 📌 Dashboards
- Overview Dashboard
- Year Comparison
- Accident Factors
- Location Analysis

## 🔍 Key Insights
- Accidents are higher in urban areas
- Most accidents are slight severity
- Traffic is a major factor

## 🛠 Tools Used
- Power BI
- DAX
- Excel
